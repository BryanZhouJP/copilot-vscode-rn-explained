# Contributing Guide

コントリビューションを歓迎します。以下のガイドラインに沿って進めてください。

## Issue の報告

- バグ報告・機能要望は [Issues](../../issues) からお気軽にどうぞ
- 既存の Issue と重複していないか確認してから作成してください

## Pull Request

1. このリポジトリを Fork する
2. 機能ブランチを作成する（例: `feat/add-new-source`、`fix/output-format`）
3. 変更をコミットする
4. Pull Request を作成する

## 変更の種類と対象ファイル

| 変更内容 | 対象ファイル |
| -------- | ----------- |
| 情報源 URL の追加・変更 | `.github/skills/vscode-rn-ai-explain/assets/sources.md` |
| レポート構成・テンプレートの変更 | `.github/skills/vscode-rn-ai-explain/references/output-format.md` |
| スキル手順の変更 | `.github/skills/vscode-rn-ai-explain/SKILL.md` |
| サンプルレポートの追加 | `reports/` ディレクトリ |

## コーディング規約（マークダウン）

- 本プロジェクトはマークダウンファイルで構成されています
- 見出しの階層は既存ファイルの構成に合わせてください
- 情報源 URL は必ず公式ドメイン（`code.visualstudio.com`、`docs.github.com`、`github.blog` 等）を使用してください
